## Chrome Extension Setup

1. Project structure

```
web-extension/
  ├── manifest.json         # MV3 manifest
  ├── sw.js                 # background service worker
  ├── content.js            # runs on Google Search page
  ├── popup.html            # optional popup
  ├── popup.js              # optional popup logic (rescan)
  └── styles.css            # badge style
```
